# Morii 音乐日记



- 在音乐选取页面，选取一段符合你当前心情的音乐

- 在创作页面，自由选取风格不同的白噪音素材进行混合

- 在编辑界面，使用文字及图片记录你当前的心情

- 在首页回顾你之前创建的音乐日记，并且将日记导出分享给他人（目前暂仅支持分享音频文件）
